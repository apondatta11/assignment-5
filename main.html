You can select an element in dom by using by getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector() for single element, querySelectorAll() for multiple elements, getElementsByName()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML returns the entire HTML content inside an element, including comments. innerText returns just the visible text content inside an element,it ignores hidden elements. textContent gets all text inside an element, even if it's hidden

Question-3: What is event delegation in the DOM?

Event Delegation allows handling events efficiently by using event bubbling. Instead of adding event listeners to multiple children elements,attaching a single event listener to a parent element and detecting which child triggered the event.

Question-4: What is event bubbling in the DOM?

Event bubbling refers to the order in which events propagate through the DOM when they are triggered. The event propagation consists of three phases: Capture Phase,Target Phase,Bubbling Phase.The capture phase occurs before the event reaches the target element.In this phase, the event starts from the root of the DOM and travels downwards through the DOM tree passing each parent element in the path of target.The target phase happens when the event reaches the target element;the element where the event is initially triggered.The bubbling phase occurs after the event has been handled by the target element.Here,the event starts from the target element and propagates upwards through its parent elements.
